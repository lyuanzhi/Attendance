# Attendance
A school attendance system built with React.

## Tech Stack
1. React

## Dependencies
```
Node.js: v18.20.2
```

## Env Setup
1. ```curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -```
2. ```sudo apt update```
3. ```sudo apt install nodejs -y```
4. ```npm install -g pnpm```
